Prework Study Guide

HTML Notes

  • The HTML page can have a head and body element
  • The head element contains information about the page
  • The body element contains elements visible to the user
  • Some elements that can be added to the head are meta and title
  • Some elments that can be added to the body are header, main, footer

CSS Notes

  • Inline style can be added with the style attribute
  • Inline css can be included in the head section using the style element
  • A margin indicates the amount of space outside an element
  • A padding indicates the amount of space around the content of an element
  • Style can be added by using the element name as a selector or a class selector
  • A class selector can be added to a css stylesheet with a .<name>

Git Notes

  • Use "git status" to see information on current branch
  • Use "git checkout -b <branch name>"" to create a new branch and switch to it
  • Use "git branch" to get a list of the branches and see what branch you are on
  • Use "git checkout <branch name>"" to swith to a branch
